
Installation file tree for SeqPup on MSDos/MSWin3
-------------------------------------------------

Use the following listing to recreate the directories and placement of
files required for SeqPup to operate properly.  The critical files and
folders are
  seqpup.exe, seqpup.pre (preferences), seqpup.hel  (help)
  [apps] folder with various .exe and .htm files
  [tables] folder with various .tab files
  
  The [seqs] folder contains example sequence data
  The [autosdat] folder contains example autosequence data
  
If your system does not have networking and Winsock software, you will need
to add a winsock.dll for seqpup to work.  See the winsock folder here.
